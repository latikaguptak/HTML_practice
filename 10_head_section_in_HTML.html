Head

it's showing the page resolution, word wrapping, SEO, SMO of the page and more.

  • "Style" tag use to give the style of the page
  • "link" tag use to give the link of the outside element on the page here mention "rel" attribute to show the relationsheep of the links "href" use for given the links
  • "meta" tag use to describe the data in the pages
    • "charset" use to set the encoding of the page.
      "name", content is basically used attributes in the name tags "viewport" use to set the resolution of the page. of the diffent devices.
